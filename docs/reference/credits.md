## Why Kolb-Bot exists

Kolby got frustrated with how complicated advanced AI tools were. Most people only know how to use basic chat models — but AI can do so much more. The problem? Everything was built for developers, written in jargon, and assumed you already knew what you were doing.

Kolb-Bot was built to fix that: an AI assistant that explains everything in simple, beginner-friendly language. Half human. Half AI. All pirate.
